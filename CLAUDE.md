# CLAUDE.md — materialcalc

Project instructions for Claude Code working in this repo. Inherits the ElevatedProgress
venture playbook from the parent folder's CLAUDE.md.

## What this is

A zero-dependency static-site generator. `generate.js` reads `data/materials.json` +
`assets/` and writes ~170 HTML pages into `public/`. It's an SEO-driven, AdSense-monetized
"how much material do I need" site: interactive calculators plus pre-computed answer pages
for mulch, topsoil, compost, gravel, river rock, sand, concrete, paint, sod, grass seed,
tile, and pavers. Target: https://materials.elevatedprogress.com/. Audience is general/US
consumers — keep the UI clean, fast, and light.

## Deploy — just push

`git push` to `main` is the deploy — GitHub Actions (`.github/workflows/deploy.yml`) runs
the generator and publishes to Pages.

- **Never manually build and commit output.** `public/` is git-ignored build output.
- **Never hand-edit anything in `public/`.**
- Commit as the neutral identity, NOT a personal name:
  `git -c user.name="materialcalc" -c user.email="materialcalc@users.noreply.github.com" commit …`

## Local build / preview

```
node generate.js     # writes ./public
node server.js       # preview at http://localhost:5056
```

On Windows set config via **PowerShell** env vars (`$env:DOMAIN='…'; node generate.js`).
Do NOT set `BASE` in Git Bash — POSIX path conversion mangles leading-slash values.

## Adding pages

Edit `data/materials.json`: `bulk` materials get a calculator + one page per area preset in
`areas`; `slabs`/`rooms` drive the concrete/paint preset pages; `seedRates`, `tileSizes`,
`paverSizes` feed the corresponding tools. Coverage math (bag yields, waste factors,
lbs/cu yd) lives in `generate.js` + `assets/tool.js` — keep those two in sync when changing
formulas, and only use standard industry coverage values.

## Don't break these (generated by generate.js, must keep serving)

- `ads.txt` + the AdSense loader in `<head>` — publisher `ca-pub-5580575158570188`.
- `sitemap.xml`, `robots.txt`, `.nojekyll`, `CNAME` (materials.elevatedprogress.com).
- A Google Search Console verification file will be added here once the property is
  verified — keep it serving after that.
